## Installation

1. Open **Extensions** in VS Code or Cursor
2. Search for **Clearline**
3. Click **Install**
4. Open the Command Palette (`Cmd+Shift+P` on macOS, `Ctrl+Shift+P` on Windows/Linux)
5. Run **Preferences: Color Theme**
6. Select **Clearline-Dark**
